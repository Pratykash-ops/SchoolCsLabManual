Computer Handling Guide

1. Folder Structure and File Naming Guidelines

Organizing your files properly helps in quick access and boosts your productivity. Here’s a recommended folder structure:

  • Documents
    • School Work
      • [Subject] (e.g., Math, Science)
      • [Project/Assignment Name]
  • Downloads (Keep temporary files and clean regularly)
  • Pictures/Videos
    • Screenshots
    • Project Media

Use clear and consistent file naming conventions:

  • Format: [Subject]_[TASK]_[Your Name]_[Date]
  • Avoid spaces, use underscores or hyphens.

2. Installed Software and Usage

The following software is installed for your academic work. Learn how to use them efficiently:

  • LibreOffice: For document creation, spreadsheets, and presentations.
  • Google Chrome: For browsing and accessing school portals.
  • VS Code: For coding and development projects.
  • PyCharm: PyCharm is special IDE for python development.
  • Anti-virus Software: Ensures the computer is secure from malware.

3. Credentials for the Computer

Protect your login details to ensure your account is secure:

  • Username/Password: Use your assigned login credentials for accessing the computer.
  • Account Recovery: Contact IT for help if you forget your password.

4. Resources for the Computer

These resources are pre-configured for learning and completing school work:

  • School Portal: Access assignments, grades, and announcements.
  • Google Classroom: Access class materials and homework.
  • Khan Academy: Use for supplemental learning in subjects like math and coding.
  • Code.org: Enhance coding skills with tutorials and projects.
  • School Library: Access research papers, eBooks, and journals via the library's portal.

5. General Computer Usage and Maintenance

Proper usage and maintenance keep the computer running smoothly and extend its lifespan:

  • Handle with Care: Shut down the system properly and avoid spills near the computer.
  • Keep Organized: Clean unnecessary files and maintain a clutter-free desktop.
  • Report Issues: Notify the lab supervisor if you encounter any technical problems.
